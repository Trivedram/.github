# Support

Thanks for using **Trivedram**! 💚

If you need help, there are several ways to get support depending on your question.

---

## 📖 Documentation

Before opening an issue, please check the project's documentation and README.

Many common questions are already answered there.

---

## 💬 Discussions

If your question is about:

* How to use a project
* Best practices
* General guidance
* Ideas or suggestions

Please open a **GitHub Discussion** instead of an Issue whenever available.

---

## 🐞 Bug Reports

If you've found a bug, please open a Bug Report using the provided issue template.

Include as much information as possible so we can reproduce the problem.

---

## ✨ Feature Requests

Have an idea?

We'd love to hear it.

Open a Feature Request and describe:

* The problem
* Your proposed solution
* Any alternatives you've considered

---

## 🔒 Security Issues

Please **do not** report security vulnerabilities publicly.

Instead, follow the instructions in our **SECURITY.md** file.

---

## ❤️ Community

Please remember that Trivedram is an open-source community.

Be respectful.

Be patient.

Help others when you can.

Together, we build better software.

---

Thank you for being part of the Trivedram community.

Happy Building! 🚀
